# FairTicket-The-anti-scalping-ticketing-platform

FairTicket 🎫
It is an anti-scalping market event ticketing system with identity-bound tickets, virtual queuing, and a scalping-proof resale marketplace.
Built as a DBMS course project. Designed around real-world problems that plague high-demand ticket sales — bots, scalpers, server crashes, and double-entry fraud.

The Problem
Every time a World Cup final, IPL playoff, or major concert goes on sale, three things happen within minutes:

Bots bulk-buy thousands of tickets before humans can
Scalpers relist the same tickets at 5–10× the original price
Servers crash under the sudden spike of millions of simultaneous requests

FairTicket is a backend-first solution to all three.
